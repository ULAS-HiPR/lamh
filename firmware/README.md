# Lamh

Lamh is for any active control like airbrakes or canards. It will control servos.

## Set-Up

```bash
git clone git@github.com:ULAS-HiPR/lamh.git
cd lamh
git submodule update --init --recursive
```

## Libraries used:

### From Braiteoiri:

- Servo
- LED

### From Comheadan:

- CAN
- PWM

### Other:

- FreeRTOS
